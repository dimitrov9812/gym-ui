# Gym business website
Live Preview - https://gim-ui.netlify.app/ (*The images are not optimised for fast page load and may render slowly*)


## Tech Stack:
   - HTML
   - SASS
   - JavaScript
   - Google Maps API
   - FontAwesome Icons

## Features: 
   - Dark Theme
   - 9 language localization (en, ch, es, ar, de, ru, fr, ja, ko)
   - Responsive Design compatible with (mobile, tablet, laptop, desktop) resolutions
